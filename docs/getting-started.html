Getting Started

  1. Run one command
npx codeandpepper/janush
  1. Choose application name
Application name? (janush-app) Awesome App
  1. Choose application type
Which application type would you like to use? (Press <space> to select, <a> to toggle all, <i> to invert selection)❯◉ Cloud ◉ Web
  1. Choose features
Which modules would you like to use? You can add this later. (Press <space> to select, <a> to toggle all, <i> to invert selection)❯◉ Authentication
Do you want add an auth emails template? (Y/n) y
